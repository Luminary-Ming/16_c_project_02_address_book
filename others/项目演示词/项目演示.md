==前端和后端通信的前提是解决跨域问题(跨域名)==

跨域是浏览器的一种安全机制, 它会阻止一个网站去另一个网站中获取资源

为什么这么设计呢, 如果说你登录了一个银行网站, 如果没有安全机制, 恶意网址就能窃取你的敏感信息(身份证号, 银行卡密码等等的), 所有为了保护上网安全, 浏览器自带这种保护机制

而我们的前端是 10.......141:5173  后端是 10.....172:8080, 就会有跨域

解决方法可以在前端, 也能在后端, 后端配置允许的域名 方法 请求头

```c
// http_server.c 中的代码
// 允许所有网站访问（* 表示通配符）
MHD_add_response_header(response, "Access-Control-Allow-Origin", "*");
// 允许的 HTTP 方法
MHD_add_response_header(response, "Access-Control-Allow-Methods", "*");
// 允许的请求头
MHD_add_response_header(response, "Access-Control-Allow-Headers", "*");
```
